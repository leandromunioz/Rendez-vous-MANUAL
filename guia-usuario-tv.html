GUÍA DE USUARIO DE TV

  • Control remoto grande para el decodificador
  • Control remoto pequeño para la televisión
  • Si el televisor indica "Sin señal", verificar que el decodificador esté encendido.
  • Con el control pequeño, presionar el botón input y seleccionar HDMI, presionar la flecha derecha para validar.
  • Seleccione los canales con el control remoto del decodificador
